splice

  • splice(array: any[], index: number, num_to_remove: number, items_to_add: any[]): void
    • Defined in modify.ts:286

    Adds and/or removes items to/from an array

    If no items_to_add are specified, then items are only removed. If num_to_remove is 0, then items are only added.

    example

    array = [10, 20, 30, 40, 50] result = Array.splice(array, 1, 3, [2.2, 3.3])

    Expected value of result is [10, 2.2, 3.2, 50].

    Parameters

    • array: any[]

      Array to splice

    • index: number

      Zero-based index at which to add/remove items. (Items are added/removed after specified index)

    • num_to_remove: number

      Number of items to remove.

    • items_to_add: any[]

      Array of items to add.

    Returns void
