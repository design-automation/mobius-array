bySlice

  • bySlice(array: any[], start: number, end: number): any[]
    • Defined in create.ts:66

    Creates a copy of a portion of an array, from start index to end index (end not included).

    example

    array = [1,2,3,4,5] result = Array.slice(array,1,3)

    Expected value of result is [2,3].

    Parameters

    • array: any[]

      Array to slice.

    • start: number

      Zero-based index at which to begin slicing. A negative index can be used, indicating an offset from the end of the sequence. If start is undefined, slice begins from index 0.

    • end: number

      Zero-based index before which to end slicing. Slice extracts up to but not including end. A negative index can be used, indicating an offset from the end of the sequence. If end is undefined, slice extracts through the end of the sequence.

    Returns any[]

    A new array.
